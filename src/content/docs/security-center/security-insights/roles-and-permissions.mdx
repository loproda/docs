Cloudflare users with the following [roles](/fundamentals/manage-members/roles/) have access to Security Insights in the Cloudflare dashboard:

- Administrator
- Administrator Read Only
- Super Administrator - All Privileges
- SSL/TLS, Caching, Performance, Page Rules, and Customization
- DNS
- Page Shield
- Page Shield Read
- Firewall
