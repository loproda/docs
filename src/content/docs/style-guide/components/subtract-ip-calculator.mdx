## `<SubtractIPCalculator>` Props

### `defaults`

**type:** `object`

An optional object containing `base` (`string`) and `exclude` (`string[]`) properties, to set default inputs.

**example:**

<SubtractIPCalculator
	client:load
	defaults={{
		base: "10.0.0.0/8",
		exclude: ["10.0.0.0/24", "10.32.0.0/11"]
	}}
/>
