Loproda se basa en estos conceptos base que se repiten a lo largo
de sus productos y funcionalidades.

## Usuarios

Es una cuenta de Loproda, esta puede ser auto-administrada por una
persona física o un sistema de terceros. Puede realizar operaciones
dentro de Loproda según los permisos y/o roles asignados.

## Organización

Un grupo estructurado de usuarios con un fin, se centra en asignar
y definir el acceso y/o las operaciones a sus usuarios miembros.

Es el centro de toda operación en Loproda.

Cada organización puede invitar a cualquier usuario dentro de Loproda
para ser parte de su organización, definiendo correspondientes
permisos y/o accesos

## Miembros

Es la participación de un usuario dentro de una Organización, esta
para definir el acceso y responsabilidad.

Al crear una cuenta en Loproda, Loproda automáticamente crea una
organización y asigna a la nueva cuenta como administrador de esa
organización.
